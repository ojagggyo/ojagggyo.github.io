# Steemit 
Steemitは、世界で最初にブロックチェーンによるブログを実現したソーシャルメディアのWebサイトです。コンテンツの公開とキュレーションに対して、暗号通貨STEEMが報酬として得られます。

# Steem Engine

STEEMベースのトークンを売買する取引所。[🔗](https://steem-engine.net/)

売買に手数料はかかりませんが、入出金に1%ずつ手数料がかかります。
